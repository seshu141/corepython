Variables:
Containers of data or place holders

Variables are also known as identifiers

Rules for variable declaration:
Variable name can be a combination  of alpha numeric and underscore
Variables can start with alphabets but numbers alone will not be taken
Special Characters and Spaces are not allowed in coding
Cases insensitive

Examples of Variables:
name= "ho"
  #This is a variable 

1name="Ho"
#This is another type of variable

Name1="HO"
#This is another type of variable

_name="hoy"

print(name) 
#prints the value of name

print(type(1name))
#tells you the class of the variable "1name"

print(id(Name1))
#tells you the id number or address of the variable "Name1"
